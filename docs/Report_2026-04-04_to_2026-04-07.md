## Architecture Decisions

| Decision | Rationale |
|----------|-----------|
| Unified `QuestionCard.astro` | Single source of truth for card rendering across 3 contexts; DRY, consistent hover/interaction |
| Client-side filtering + pagination (B8) | All 152 cards in DOM with `display:none` toggling; avoids page reloads, filter resets pagination |
| Event-delegated tag click handler | Tags inside card `<a>` wrappers need `stopPropagation` to navigate to tag archive instead of question page |
| `:global()` inside scoped parent | `set:html` content doesn't receive Astro scoped data attributes; `:global()` wrapped in a scoped parent selector targets it safely |
| `slugify()` inside `getStaticPaths()` | Astro hoists `getStaticPaths` to isolated module scope; top-level imports not available |
| Sequential Notion DB fetches | Parallel fetches triggered rate limits; sequential with exponential backoff is reliable |
| `.cache/notion/` layer | Avoids re-fetching 152 questions + blocks on every dev server restart |

## Key Bug Fixes

| Bug | Root Cause | Fix |
|-----|-----------|-----|
| `slugify is not defined` in tag archive | Astro `getStaticPaths` runs in isolated module scope | Moved function inside `getStaticPaths()` |
| Tags don't change colour on hover | `set:html` content lacks Astro scoped attributes | `:global()` wrapped in scoped parent |
| All categories showing 0 questions | Notion property is `Category`, not `Okay to Ask Category` | Fixed field name in `fetchers.ts` |
| Notion rate limit errors | Parallel DB queries hitting 429s | Sequential fetches + batched block requests |
| Simple Mode not toggling on glossary | CSS selector not wrapped in `:global()` | Added `:global()` wrapper |
| Glossary tooltip not matching plurals | Regex only matched exact term | Added inflection suffixes to pattern |
